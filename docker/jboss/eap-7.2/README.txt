* Below the instructions about the current image and files in this folder.

** Dockerfile: This is the steps to build the image with jboss eap 7.

** help-commands.txt: This file contains the commands to build and run the dockerfile.

** docker-compose.yml: This file up the current image(jboss eap 7), expose ports and to the simple deploy of .war to sample.

** target(folder): This folder contains the .war to compose do the deploy sample.
